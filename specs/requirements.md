# System Requirements

This project focuses on monitoring and logging mechanisms in open source
operating systems, with an emphasis on DevOps and SecOps use cases.

## Functional Requirements
- Monitor running system services and processes
- Collect and display system log entries
- Support basic service status checks
- Provide readable output for security analysis

## Non-Functional Requirements
- Lightweight scripts with minimal dependencies
- Compatibility with common Linux distributions and Windows
- Clear and simple output for educational purposes

## Platform Requirements
- Linux: Ubuntu / Debian (bash)
- Windows: Windows
